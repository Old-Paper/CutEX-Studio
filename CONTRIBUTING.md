# Contributing to CutEX Studio

感谢你改进 CutEX Studio。提交改动前请：

1. 使用 Node.js 20 或更高版本，并安装 FFmpeg/FFprobe 6 或更高版本。
2. 从 `main` 创建功能分支。
3. 运行 `pnpm install --frozen-lockfile` 安装依赖。
4. 确保 `pnpm test`、`pnpm lint`、`pnpm typecheck` 和 `pnpm build` 全部通过。
5. Pull Request 中说明改动目的、用户影响和验证方式。

请勿提交本地媒体、导出文件、日志、项目文件、密钥或访问令牌。
